# Contributing

Thanks for contributing to **ai-llm-evals-lab**.

## Development setup
1. Create a virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run tests with `python -m pytest -q`.

## Pull request checklist
- [ ] Tests pass locally
- [ ] New behavior is documented in README or docs
- [ ] Data files are validated for format consistency
- [ ] Commit messages are clear and scoped
